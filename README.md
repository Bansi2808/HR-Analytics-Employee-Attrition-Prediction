## 🚀 Overview

This project is an end-to-end **Machine Learning web application** that predicts employee attrition based on HR data.

It combines:
- 🤖 Predictive Modeling  
- 🎯 Interactive UI  
- 🔐 Role-Based Authentication (RBAC)  

---

## 🔐 Key Features

- 🔑 User Authentication (Login & Signup)  
- 👥 Role-Based Access Control (RBAC)  
- 🤖 Machine Learning Prediction Model  
- 📊 Real-time Risk Score Visualization  
- ⚠️ Business Recommendations for HR  
- 🌐 Deployed Streamlit Application  

---

## 🧠 Problem Statement

Employee attrition is a major challenge for organizations.

This application helps HR teams to:
- Identify employees at risk of leaving  
- Take proactive retention actions  
- Make data-driven decisions  

---

## 🛠️ Tech Stack

- Python  
- Pandas, NumPy  
- Scikit-learn  
- Streamlit  
- Joblib  
- bcrypt (for secure authentication)  

---

## 📊 Input Features

- Satisfaction Level  
- Last Evaluation Score  
- Number of Projects  
- Monthly Working Hours  
- Years at Company  
- Promotion Status  
- Department  
- Salary Level  

---

## 📈 Output

- ✅ Prediction: Will employee leave or not  
- 📊 Risk Score (Probability of attrition)  
- ⚠️ Business recommendation  

---

## 🔐 Authentication System

- Signup & Login functionality  
- Password hashing using bcrypt  
- Session-based authentication  
- Role-based access (Admin/User)
